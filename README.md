# J case 接案提案平台

### DEMO 
- 前台 - https://momolly1024.github.io/Jcase_project/
- 後台 - https://momolly1024.github.io/Jcase_project/#/back

### 開發工具、部屬環境
- Vue Cli
- BootstrapVue
- Axios 前後台串接
- Node.js 
- MongoDB
- 前台放Github，使用Travis CI自動打包工具
- 後端架在heroku commit後deploy
